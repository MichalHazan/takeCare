-------Review---
POST /api/reviews
Body:
{
    "professionalId": "professionalObjectId",
    "rating": 5,
    "comment": "Excellent service!"
}

Get Reviews for a Professional:
GET /api/reviews/:professionalId

Update a Review:
PUT /api/reviews/:reviewId
Body:
{
    "rating": 4,
    "comment": "Updated comment"
}

Delete a Review
DELETE /api/reviews/:reviewId
